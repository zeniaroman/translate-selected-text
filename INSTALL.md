# Инструкция по установке

## Шаг 1: Создание иконок

Перед установкой расширения необходимо создать иконки. Выберите один из способов:

### Способ 1: Использование HTML генератора (Рекомендуется)

1. Откройте файл `icons/generate-icons.html` в браузере Chrome
2. На странице появятся 3 ссылки для скачивания
3. Скачайте все три файла: `icon16.png`, `icon48.png`, `icon128.png`
4. Поместите скачанные файлы в папку `icons/`

### Способ 2: Конвертация SVG

1. Откройте онлайн конвертер (например, https://cloudconvert.com/svg-to-png)
2. Загрузите файл `icons/icon.svg`
3. Создайте три версии с размерами: 16x16, 48x48, 128x128 пикселей
4. Сохраните как `icon16.png`, `icon48.png`, `icon128.png` в папку `icons/`

### Способ 3: Использование Node.js скрипта

```bash
node create-icons.js
```

Это создаст простые placeholder иконки (синие квадраты).

## Шаг 2: Установка расширения в Chrome

1. Откройте Chrome и перейдите на `chrome://extensions/`
2. Включите **"Режим разработчика"** (Developer mode) в правом верхнем углу
3. Нажмите кнопку **"Загрузить распакованное расширение"** (Load unpacked)
4. Выберите папку проекта `translate-selected-text`
5. Расширение появится в списке установленных расширений

## Шаг 3: Проверка работы

1. Откройте любой сайт с текстом
2. Выделите слово или фразу
3. Нажмите **⌘ Command+Z** (Mac) или **Ctrl+Z** (Windows/Linux)
4. Должно появиться всплывающее окно с переводом

## Настройка языков

1. Нажмите на иконку расширения в панели инструментов Chrome
2. Выберите нужные языки в выпадающих списках
3. Нажмите **"Save Settings"**

## Возможные проблемы

### Ошибка при загрузке расширения
- **Причина**: Отсутствуют файлы иконок
- **Решение**: Выполните Шаг 1 для создания иконок

### Расширение не реагирует на Command+Z
- Убедитесь, что текст выделен перед нажатием горячей клавиши
- Проверьте, что другие расширения не перехватывают эту комбинацию
- Перезагрузите страницу после установки расширения

### Перевод не работает
- Проверьте подключение к интернету
- Откройте консоль разработчика (F12) и проверьте наличие ошибок
- Убедитесь, что в настройках выбраны разные языки для перевода
