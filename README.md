# Translate Selected Text - Chrome Extension

> **Личное расширение для изучения языков и перевода текста**  
> Создано для личного использования. Если вам будет полезно - доработайте под себя.

### Базовое использование
1. Выделите любой текст на странице
2. Нажмите `⌘ Command+Z`
3. Перевод появится во всплывающем окне

## 📋 Обзор функционала

Это Chrome расширение объединяет в себе несколько полезных инструментов для изучения языков и перевода текста:

### 🔄 Перевод выделенного текста
- **Горячая клавиша**: `⌘ Command+Z` (Mac) или `Ctrl+Z` (Windows/Linux)
- **Всплывающее окно**: Перевод появляется рядом с выделенным текстом
- **Быстрое копирование**: Одним кликом скопировать перевод
- **Поддерживаемые языки**: Русский, английский, испанский, французский, немецкий, китайский

### 📚 Словарь и история переводов
- **Сохранение переводов**: Автоматическое сохранение всех переводов
- **Категоризация**: Возможность группировать слова по темам
- **Изображения**: Добавление визуальных ассоциаций к словам
- **Поиск и фильтрация**: Быстрый поиск по словарю
- **Теги**: Организация слов по тегам для лучшего запоминания

### 🎓 Курсы изучения языков
- **Методология Эрика Гуннемарка**: Изучение базового минимума языка
- **Английский язык**: Курс с грамматикой и фразами
- **Испанский язык**: Аналогичный курс для испанского
- **Интерактивные упражнения**: Практика с проверкой ответов(очень базово)

### 📖 Грамматические курсы
- **Теория**: Объяснения грамматических правил
- **Таблицы спряжения глаголов**: Интерактивные таблицы по временам(только испанский)
- **Упражнения**: Практические задания с проверкой
- **Прогресс**: Отслеживание пройденных упражнений

### 💬 Полезные фразы
- **Разговорные фразы**: Самые употребительные выражения
- **Тематические подборки**: Фразы по ситуациям (приветствия, покупки, ресторан)
- **Аудио**: Возможность прослушать произношение
- **Карточки**: Система для запоминания фраз

## 🚀 Установка

### Для разработчиков
1. Откройте Chrome: `chrome://extensions/`
2. Включите **Режим разработчика**
3. Нажмите **Загрузить распакованное расширение**
4. Выберите папку проекта `translate-selected-text`

---

## 🎯 Как использовать

### Базовый перевод
1. Выделите любой текст на странице
2. Нажмите `⌘ Command+Z`
3. Перевод появится во всплывающем окне

### Настройки перевода
1. Нажмите на иконку расширения
2. Выберите исходный и целевой языки
3. Сохранение выбора происходит автоматически

### Изучение языков
1. Откройте словарь через popup расширения
2. Выберите язык изучения
3. Пройдите грамматический курс или изучите фразы

### Словарь
1. Все переводы автоматически сохраняются
2. Добавляйте теги и категории
3. Используйте поиск для быстрого доступа

## 🔧 Технические особенности

### Используемые технологии
- **Manifest V3**: Современный стандарт Chrome расширений
- **Google Translate API**: Публичный endpoint для переводов
- **Chrome Storage API**: Синхронизация настроек и локальное хранение
- **Vanilla JavaScript**: Без зависимостей, чистый код

### Хранение данных
- **chrome.storage.sync**: Настройки языков (синхронизация)
- **chrome.storage.local**: История переводов и словарь (локально)

### Поддерживаемые языки
- 🇷🇺 Русский (ru)
- 🇬🇧 English (en) 
- 🇪🇸 Español (es)
- 🇫🇷 Français (fr)
- 🇩🇪 Deutsch (de)
- 🇨🇳 中文 (zh-CN)
- Можно дополнять
